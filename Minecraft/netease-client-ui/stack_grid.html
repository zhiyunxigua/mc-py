stack_grid

作者：西瓜

介绍

stack_grid就是一维的网格，可以选择扩展方向是竖直还是水平，并且可以在python层绑定网格数量

开始编写

json示例

Example.json 

    "stack_grid_example": {
        "type": "stack_grid",
        "bindings": [
            {
                "binding_condition": "always",
                "binding_name": "#stack_grid_example.item_count", // 我们通过该变量名复写引擎中对应的变量名
                "binding_name_override": "#StackGridItemsCount" // 这是引擎中的变量名，用于控制stack_grid的数量
            }
        ],
        "collection_name": "itemToggles", // 绑定的集合名
        "controls": [
            {
                "item_button@stack_grid_item_button" : {} // stack_grid中的template，改变#StackGridItemsCount的值会动态克隆所有controls
            }
        ],
        "item_count": 10,  // 默认值
        "orientation": "vertical", // 扩展方向，垂直
        "property_bag": {  // 属性包
            "#itemToggleGrid.item_count": 10 // 默认有10个item，实际根据绑定值返回实时更新
        },
        "size": [ "100%", "default" ]
    }

python示例

绑定stack_grid内容的数量

Example.python 
    @ViewBinder.binding(ViewBinder.BF_BindInt, "#stack_grid_example.item_count")
    def OnReturnStackGridCount(self):
        # 返回stack_grid内容的数量，对应json的#stack_grid_example.item_count变量
        return 10
