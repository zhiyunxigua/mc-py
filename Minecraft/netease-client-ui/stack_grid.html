介绍

stack_grid就是一维的网格，可以选择扩展方向是竖直还是水平，并且可以在python层绑定网格数量

demo下载
开始编写

json示例

这是一个基本的stack_grid示例，controls就是stack_grid里面的控件，改变#StackGridItemsCount就会克隆所有的controls

Example.json 
    "stack_grid_example": {
        "type": "stack_grid",
        "bindings": [  // 绑定，对应的python代可以看下面的python示例eg.1
            {
                "binding_condition": "always",
                "binding_name": "#stack_grid_example.item_count", // 我们通过该变量名复写引擎中对应的变量名
                "binding_name_override": "#StackGridItemsCount" // 这是引擎中的变量名，用于控制stack_grid的数量
            }
        ],
        "collection_name": "itemToggles", // 绑定的集合名
        "controls": [
            {
                "item_button@stack_grid_item_button" : {} // stack_grid中的template，改变#StackGridItemsCount的值会动态克隆所有controls
            }
        ],
        "item_count": 10,  // 默认值
        "orientation": "vertical", // 扩展方向，垂直
        "property_bag": {  // 属性包
            "#itemToggleGrid.item_count": 10 // 默认有10个item，实际根据绑定值返回实时更新
        },
        "size": [ "100%", "default" ]
    }

接下来，我们需要实现按钮按下之后调用python方法和在python层设置按钮显示文本

现实性按钮按下后调用python方法，这里我使用button_mappings（按钮映射）的方式

首先修改按钮里面button_mappings的内容

Example.json 
"button_mappings": [
    {
        "from_button_id": "button.menu_select",  // 按钮id，这个表示鼠标左键点击
        "to_button_id": "#button_click",  // python层绑定的名字
        "mapping_type": "pressed"  // 映射类型，这里表示按钮按下时触发
    },
    {  // 可以同时存在多个映射，这里只需要一个映射就可以
        "from_button_id": "xxx",
        "to_button_id": "#xxx",
        "mapping_type": "xxx"
    }
]

然后绑定按钮的显示文本


按钮模板里面有一个button_label控件，我们可以直接修改这个控件

在button_label里面添加绑定，这里我们使用集合绑定

Example.json 
"button_label@netease_editor_template_namespace.button_label": {
    "text": "#button_text",
    "bindings": [
        {
            "binding_name": "#button_text", // 绑定名字，引擎会去找Python里的#button_text
            "binding_type": "collection", // 绑定类型，默认global，collection为集合绑定
            "binding_collection_name": "stack_grid_group",  // 绑定的集合名字
            "binding_condition": "visible"  // 绑定条件，visible为当可见时触发一次
            }
    ]
}

这样，我们的json就算完成了


python示例

eg.1 绑定stack_grid内容的数量

Example.py 
    @ViewBinder.binding(ViewBinder.BF_BindInt, "#stack_grid_example.item_count")
    def OnReturnStackGridCount(self):
        # 返回stack_grid内容的数量，对应json的#stack_grid_example.item_count变量
        return 10

eg.2 绑定按钮映射

Example.py 
    # 按下时通过re函数计算出按下的是滚动列表中的第几个按钮
    @ViewBinder.binding(ViewBinder.BF_ButtonClickUp, "#button_click")
    def OnButtonClick(self, args):
        # 按钮按下后有个ButtonPath参数是按钮的路径，可以通过路径的名字，获取按钮的index
        # 获取到的路径是字符串，通过re函数计算之后的数据类型是list[str]，所以要先索引0，然后转成int格式
        # 获取到的index是从1开始的，如果需要从0开始，就要-1
        button_index = int(re.compile(r'\d+').findall(args["ButtonPath"].split('/')[-1])[0]) - 1
        print "第 §b" + str(button_index) + "§8 个按钮按下啦！"
